# Cozy-Fragments

_Forkable, fun, and cozy_

**Wanted terminal fragments that provide:**

- base configuration
- targets
- profiles
- a cozy bit of flair

## Want a nice colour theme?

Use Twilight - https://github.com/vegcom/WindowsTerminal-Twilite

```jsonc
    // Colours
    "colorScheme": "Twilite",
    "unfocusedAppearance": {
        "colorScheme": "Twilite (Darker)"
    },
```
